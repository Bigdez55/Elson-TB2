## Project Organization

The project is organized into the following main directories:

- `Elson/backend/` - FastAPI backend server
- `Elson/frontend/` - React frontend application
- `Elson/trading_engine/` - Trading engine with ML models
- `Elson/infrastructure/` - Kubernetes and deployment files
- `Elson/beta-release-setup/` - Beta release configuration and scripts
- `docs/` - Documentation files
